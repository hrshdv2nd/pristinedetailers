# Pristine Detailers

Melbourne's premier mobile car detailing service website.

## Pages

- `index.html` — Home
- `services.html` — Services
- `about.html` — About
- `contact.html` — Contact
- `login.html` — Customer / Detailer / Admin portal

## Deploy (GitHub Pages)

1. Push these files to the `main` branch
2. Go to Settings → Pages
3. Source: Deploy from branch · Branch: `main` · Folder: `/ (root)`
4. Site goes live at `https://hrshdv2nd.github.io/pristinedetailersconnect/`
